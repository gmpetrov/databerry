description: 'Make your data ready for large language models'
---

Chaindesk Datastores allows you to easily import your custom data and make it ready to be used by AI Agents.
In order to train agent on custom data follow fhe following steps

<Steps>
  <Step title="Create an Agent">Follow [Getting Started guide](/agent)</Step>
  <Step title="Add Knowledge">
    Create a [Datastore](https://app.chaindesk.ai/datastores) and connect it to
    your Agent with the Datastore Tool.
  </Step>
  <Step title="Connect Datastore to an Agent">
    From your Agent settings tab > Toosl section, setup a `Datastore Tool`
  </Step>
</Steps>
