## FAQ

### How to install a private plugin?

Installing a private is straightforward. Just follow the video above in order to install an unofficial Plugin

### How to publish the plugin to make it available to all ChatGPT users?

In order to publish a plugin, you need to submit a request through the plugin submission bot.
**Note that your datastore needs to be public in order to make the plugin available to all ChatGPT users.**

[Learn more](https://platform.openai.com/docs/plugins/review/submit-a-plugin-for-review)

### The plugin is not picked-up by ChatGPT

If the plugin is not picked-up by ChatGPT you might need to optimize the "description for humans" field of your plugin.

[Plugin description best practices](https://platform.openai.com/docs/plugins/getting-started/writing-descriptions)
