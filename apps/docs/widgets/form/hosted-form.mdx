description: 'Hosted Chaindesk Form'
---

All forms are available from Chaindesk's Hosted URL.

```bash
https://app.chaindesk.ai/forms/YOUR_FORM_ID
```

<Note>
  You can find the Form ID in the Form settings under the Settings tab.
</Note>

The easiest way to get started, is to simply redirect the user to this page from within your app or website.
