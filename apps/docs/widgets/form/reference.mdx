## Attributes / Props

<Tabs>
  <Tab title="HTML" >
    ```html
    <script type="module">
  import Form from 'https://cdn.jsdelivr.net/npm/@chaindesk/embeds@latest/dist/form/index.js';

Form.initStandard({
  formId: 'YOUR_AGENT_ID',

  onEnd: function (values) {
    console.log('Form submitted with values:', values);
  },
})


</script>
    ```

<ParamField path="formId" type="string" required>
  ID of the Agent
</ParamField>


<ParamField path="onEnd" type="function">
  Callback function that is called when the form is submitted. The function receives the form values as an object.
```js
const form = await Form.initBubble({
  agentId: 'YOUR_AGENT_ID',
  onEnd: function (values) {
    console.log('Form submitted with values:', values);
  },
})
````
