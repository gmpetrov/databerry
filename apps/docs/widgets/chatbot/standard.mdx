---
title: 'Standard'
description: 'Install on your website'
---

To install an Agent on your website using the Standard widget use the follwing code snippet:

```html
<script type="module">
  import Chatbox from 'https://cdn.jsdelivr.net/npm/@chaindesk/embeds@latest/dist/chatbox/index.js';

  Chatbox.initStandard({
    agentId: 'YOUR_AGENT_ID',
  });
</script>

<chaindesk-chatbox-standard />
```

Replace `YOUR_AGENT_ID` with the ID of your Agent

<Note>
  You can find the Agent ID in the Agent settings under the Settings tab.
</Note>

See all the available attributes in the [Chatbot Reference](/widgets/chatbot/reference).
